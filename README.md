# WebInspector — Анализатор сайтов

**WebInspector** — это одностраничное веб-приложение для базового анализа сайтов по URL.  
Приложение проверяет страницу, собирает основные SEO-метрики и выводит рекомендации по улучшению сайта.

Проект выполнен на **HTML, CSS и JavaScript** без сторонних фреймворков.

---

## Возможности проекта

Приложение позволяет:

- вводить URL сайта для проверки;
- автоматически добавлять `https://`, если протокол не указан;
- проверять корректность введенного адреса;
- получать HTML страницы через прокси-сервис;
- отображать визуальный предпросмотр сайта;
- анализировать SEO-параметры страницы;
- формировать отчет с ошибками, предупреждениями и успешными проверками;
- сохранять историю проверок в `localStorage`;
- загружать результаты из истории;
- очищать историю проверок;
- переключать светлую и темную тему;
- копировать отчет в буфер обмена;
- экспортировать отчет через печать или сохранение в PDF.
